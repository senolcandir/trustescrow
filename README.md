# TrustEscrow — Secure USDC Payments, Built on Arc

A decentralized escrow payment platform built on Arc Testnet. TrustEscrow
doesn't sell products, host listings, or connect buyers and sellers — it
only secures the USDC payment between two parties who already agreed on a
deal elsewhere (Facebook Marketplace, Telegram, Instagram, etc.).

## Structure

- `contract/` — Solidity smart contract (Hardhat), deployed on Arc Testnet
- `web/` — Next.js web interface, live at https://trustescrow-web.vercel.app
